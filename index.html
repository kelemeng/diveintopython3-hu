Ugorj fejest a Python 3-ba

Az Ugorj fejest a Python 3-ba bemutatja a Python 3 nyelvet és eltéréseit a Python 2-től. A Dive Into Python könyvhöz képest körülbelül 20% átdolgozott és 80% új anyagot tartalmaz. A könyv most teljes, de a visszajelzésnek mindig örülünk.
